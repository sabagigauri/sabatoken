About SabaToken

SabaToken (SABA) is a custom ERC-20 token built using the OpenZeppelin smart contract library and deployed on the Ethereum Sepolia testnet. It was created as part of a blockchain development assignment to demonstrate the full lifecycle of token creation — from writing a Solidity smart contract to deploying it live using Hardhat and Alchemy.

The token has a fixed initial supply of 1,000,000 SABA, all minted to the deployer's wallet at launch. The contract owner can also mint additional tokens as needed using the built-in mint function.
